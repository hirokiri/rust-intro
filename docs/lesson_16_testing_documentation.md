Lesson 16 — Testing & Documentation

Overview

Covers writing unit and integration tests, doc tests, and writing good documentation comments (///). Explain test attributes, cargo test, and running selective tests.

Exercises

1. Write unit tests for edge cases of simple functions in src/answers.
2. Add a doctest to a function's doc comment and run cargo test.
3. Create an integration test under tests/ that uses the public API.
