Lesson 17 — Macros & Advanced Features

Overview

Covers macro_rules! declarative macros, a brief intro to procedural macros (derive/attribute), and other advanced language features.

Examples

- macro_rules! examples (variadic patterns, repetition)
- Using closures, iterators, and higher-ranked trait bounds briefly

Exercises

1. Write a small macro that implements a debug-print wrapper.
2. Read about procedural macros and try a derive macro crate example (optional).
